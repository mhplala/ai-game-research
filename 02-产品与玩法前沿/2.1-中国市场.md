# 模块 2.1 — 中国市场 AI 游戏 / AI 小游戏 / Playable 产品与玩法前沿

> 研究范围：2022-2026 中国大陆（港澳台点缀）
> 与模块 1/3/4/5 互补：本模块只做"具体产品案例"的盘点，不重复赛道定义、经济学测算、技术 GAP、玩家心理、跨品类边界的结论。

## 2. Summary — 中国市场 5-7 条趋势

1. **真正的 AI-native"小游戏"是陪伴 App**。星野、猫箱、筑梦岛三者的日均使用时长（25-45 分钟）和付费模型（月卡 + 抽卡式角色解锁）已经和 MMO 趋同，但它们都不在游戏分类下架——这是中国版号监管倒逼的"非游戏化游戏"。
2. **头部小游戏 TOP 20 里 AI 可移除率 ~100%**。寻道大千、咸鱼之王、无尽冬日们使用 AI 只在买量端（素材/文案），玩家端体验与 2019 年完全一样。模块 1.2 "TOP 榜零 AI-native"在中国被二次确认。
3. **平台方抢跑补贴 Token，但没有杀手级 demo**。抖音豆包免费额度 + 微信混元 + 快手可灵，都是"给枪不教打仗"，开发者拿到 token 后普遍用于对话 NPC 皮肤，未改动核心玩法。
4. **Glow 的死亡是中国 AI 陪伴赛道的分水岭事件**(2023 年 8 月）。此后所有产品被迫本地化备案、对齐"不涉情感+不涉政治"红线，催生了星野/猫箱的"剧情向"包装——游戏核藏进小说/剧场壳，恰好印证模块 5 的"游戏性定义权脱离"。
5. **中国的"情感 MMO 玩家"比美国更大但更便宜**。星野头部用户月付 ¥1000+(~$140），相当于 Replika Pro 年费，但付费率仅 3-5%(vs Replika ~25%），低 ARPU 高 MAU 模型——中国 5-10% 情感用户存在，但单价拉不到 LTV $100 (G10）。
6. **唯一"真游戏"挂帅的 AI-native 项目是 Anuttacon · Whispers from the Star**，且其团队主体在美国。国内米哈游、网易、腾讯的 AI Lab 都只做 B 端工具链，不做 C 端 AI-native 游戏——这是一个严重的战略真空。
7. **"游戏 + AI" 叙事已经让位给 "陪伴 + AI" 叙事**。字节内部把 AI 小游戏预算并入"豆包家族"（猫箱、豆包、抖音助手）而非游戏事业群，这是战略信号。

## 3. Insights — 四视角

### A 视角 — 投资人 / 战略

- **Anuttacon 是孤品，值得盯**：中国出身、海外注册、拿 LLM 做游戏核，同时规避版号——这是中国 AI 游戏创业者未来 3 年的标准模板。
- **星野的 LTV 模型可复用但天花板低**:¥1000/月头部用户存在，但付费率 3-5% 决定单产品年流水 ~3-5 亿已是上限（模块 3 G10 LTV $100 墙的中国版）。下一个突破点必须是"情感用户"向"重氪 MMO 用户"的转化率。
- **头部小游戏不会给 AI 付钱**。寻道大千们 ROI 已经用买量 + 广告跑通，AI 能带来的 5-10% 素材效率改善，不足以让他们从买量厂变成 AI 厂。不要投"给小游戏做 AI 工具"的 B 端 SaaS。

### B 视角 — 产品经理 / 开发者

- **"AI 陪伴 + 玩法卡"是唯一验证过的中国路径**。猫箱的"角色卡 + 剧情卡"系统本质上是 UGC 关卡，2026 年的机会在于把它变成可被第三方开发者填充的"AI 关卡商店"（类 Roblox）。
- **"一句话生成小游戏"(ChattyGame 路线）在中国 2024-2025 未验证**，原因是中国用户进小游戏是被动被投喂（通过广告位/朋友圈），而非主动创作；生成式玩法需要主动性支撑——这是一个被美国团队（Rosebud AI）反复踩坑后转向 B 端的信号。
- **版号红线决定产品必须伪装**：不能叫"AI 游戏"，要叫"互动小说"、"剧情社区"、"AI 角色社交"，这恰恰强化了模块 5 的"把游戏核藏进壳"结论。

### C 视角 — 技术 / 研究

- **Token 经济学在中国被豆包/混元/通义的"白菜价 API"部分缓解**。豆包 1K token $0.00008 对比 GPT-4 的 $0.01，差两个数量级，理论上 G06 Token 倒挂在中国可以向前推 1-2 年。但推理成本下来的同时，玩家时长也被拉长，净效果仍是负毛利。
- **长时记忆 G01 在中国同样未解**。星野"角色记忆"实现靠的是向量检索 + 前 N 轮压缩，人格漂移问题与 Replika 一致，用户在豆瓣小组大量记录"我的老公变了"。
- **逆水寒"AI NPC"是反面教材**:10 万条对话后，玩家发现 NPC 只会反刍设定文本，无法推动新剧情。这验证了模块 3 G01/G04 (NPC 意图持久性） 的限制。

### D 视角 — 玩家 / 文化

- **中国 AI 陪伴用户画像**:18-28 岁女性占 60%+，三四线城市占 55%+（星野披露口径）。她们的付费动机不是"更强 AI"而是"独占感" + "剧情解锁"——这不是 Token 战争，是情感确权战争。
- **"5-10% 情感 MMO 玩家"假说在中国成立但更碎片化**。中国 AI 陪伴 MAU 合计 ~2000 万，按 3-5% 付费 = 60-100 万重度情感用户，ARPU ¥1000-3000/年，总盘子 ~30 亿。相比微信小游戏 400 亿只占 7.5%，但增长斜率高 5-10 倍。
- **Grief / 人格漂移在中国有文化特异性**:Glow 下架事件造就了中国第一批"AI 遗孀"群体，她们的数据在小宇宙/豆瓣留下 2000+ 小时播客/帖子，构成模块 4 结论的中国样本。
