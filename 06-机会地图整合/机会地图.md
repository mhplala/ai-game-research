## §0. TL;DR(200 字）

**核心判断**:Runtime AI × IAA 小游戏是经济学真空带（模块 1.1 + 3 · G06 Token 倒挂），但 **Generative Playable Feed(GPF)** 子品类用 **Pipeline AI + UGC 飞轮** 从根本上把 Runtime 成本归零，是 2025-2026 观察到唯一同时解决 Token 经济学、内容枯竭、冷启动三大痛点的形态（模块 2.5）。**最大机会**是"中国版 GPF"——抖音/微信/小红书生态中的 Pipeline-Creator 象限全球仅 7 家、中国零玩家，12-24 个月后被 YouTube Playables Builder / Roblox / MWM 平台化吃掉（模块 1.4 · 2.5）。**最应规避**的是正面 Runtime AI × IAA 交集、纯 C 端 AI 陪伴独立公司（Character.AI $2.7B 退出是"失败的成功退出"）、AAA 内嵌 AI NPC（逆水寒反面）与 AI-washed 叙事型公司。

**一句话版**:GPF 是唯一真实的窗口，中国版 Pipeline-Creator 是最大空白，12-24 个月内必须下注。

## §1. 主命题（深化）

### 1.1 修订后的主命题

> **"Runtime AI × 小游戏是真空带（Token 经济学卡死），但 Generative Playable Feed(GPF）子品类正在用 Pipeline AI + UGC 飞轮打破它 — 窗口期 12-24 个月，之后被 YouTube/TikTok/Roblox 平台化吃掉。"**

### 1.2 为什么 Runtime 真空带成立（三重论证）

1. **经济学根因**:GPT-4o-mini 单会话 Token 成本 ~$0.014/小时，而小游戏 IAA ARPDAU < $0.05,AI 税直接吃掉 15-40% 毛利【模块 1.1 + 1.3 + 3 · G06】
2. **退出模式印证**:Character.AI 28M MAU → 被 Google $2.7B "失败的成功"授权，Inflection → Microsoft $650M reverse acqui-hire,**C 端独立 IPO 路径在 2024 正式关闭**【模块 2.2】
3. **TOP 榜单零信号**:2024 Q2 微信小游戏 240+ 款季度千万流水榜单中零 AI-native 作品，TOP 1 寻道大千仅用 AI 做买量素材【模块 1.1 · 2.1】

### 1.3 为什么 GPF 是唯一可行破局

**Pipeline AI** 的经济学方程式：
- **AI 只在创作时被调用一次**，消费端零 Token 成本
- 单次生成 ~$0.1-0.5，靠 **creator : consumer ≥ 1 : 100** 的 UGC 杠杆摊薄
- 消费端回归传统 IAA / 订阅，**AI 税归零**【模块 2.5】

**一对一映射每个模块的命题**:
| 模块 | 命题 | GPF 如何解 |
|---|---|---|
| 1.1 双刀 | AI-native × Runtime 象限红海 42 家 | GPF 在 **Pipeline-Creator** 象限全球仅 7 家，中国零 |
| 1.3 平台 | AI 税 15-40% ARPDAU | Pipeline AI 把 AI 税搬到创作端 |
| 3 · G06 | Token 倒挂 | GPF 唯一不受 G06 影响的形态 |
| 4 · 情感 MMO | D7-D30 断崖仅服务 5-10% | GPF 服务轻度 95%，反面形态 |
| 5 · 定义权脱离 | 游戏壳 ≠ 游戏 | GPF = 定义权脱离的终极形态，做 playable 但不叫游戏 |

### 1.4 双刀分类框架（本文全文复用）

**刀 1 · AI 可移除性**:**AI-native**（移除 AI 即死）/ **AI-augmented**(AI 让体验更好）/ **AI-washed**(AI 只是 PR)/ **Enabler**(AI 基础设施 B 端）
**刀 2 · AI 存在位置**:**Runtime**（在线推理）/ **Pipeline-Dev**（开发工具）/ **Pipeline-Creator**（用户创作）/ **Pipeline-Embedded**（素材预生成）/ **Hybrid**

## §4. 确定性机会（5 个强判断）

### O1. 中国版 GPF(Generative Playable Feed)💎 **P0**

**服务视角**:A 产品 / B 投资 / D 战略

**机会描述（200 字）**：在抖音 / 微信 / 小红书 / B 站任一生态内，做一款"用户一句 prompt → AI 生成 playable → feed 消费 → remix 飞轮"的独立 App 或小游戏。底层用豆包 / Hunyuan / 通义千问的 pipeline 生成能力，**AI 只在创作时被调用一次**，消费端回到传统 IAA 或虚拟币。对标全球玩家 MWM Rezona / LoopIt / YouTube Playables Builder / Upit。核心差异化：中文 meme / 本土 IP / 抖音式分发。这是 Pipeline-Creator × AI-native 象限全球仅 7 家、中国零玩家的空白位（模块 1.4）。

**多模块佐证**:
- 模块 2.5 GPF 品类定义：Pipeline AI + UGC feed + creator:consumer ≥1:100 【S-2.5-*】
- 模块 1.4 双刀统计：Pipeline-Creator 全球 7 家 / 中国零 【S-1.4-*】
- 模块 1.3:AI 税 15-40% ARPDAU → GPF 把 AI 税搬到创作端 【S-1.3-*】
- 模块 3 · G06:Token 倒挂根本解 【S-3-G06】
- 模块 5：游戏性定义权脱离的终极形态 【S-5-T1】

**为什么"确定性"(3 论据）**:
1. **同时解决三大痛点**(Token + 内容 + 冷启动）— 是 Phase 1-3 中**唯一**同时解三大痛点的形态
2. **全球已经有 6+ 个玩家在跑**(MWM 双品牌 + YouTube 官方 + Rosebud + Upit + Replit + Websim），品类已被验证不是假设
3. **中国 Pipeline-Creator 象限字节 / 腾讯 / 米哈游都缺席**，招聘与专利侧面验证"正在筹备但未出货"（模块 2.5 中国 gap)

**TAM 估算**：若能吃到抖音小游戏大盘 10-20%，对应 **¥15-40 亿流水**（基于 1.2 抖音小游戏 150-200 亿估算），上限被创作者群体规模和 feed 时长决定。

**可复用路径（"四步曲"概念）**:
1. **第一步：品类占位**(Week 1-8)MVP 上线，只做一类 prompt（如"生成一个 idle 放置")
2. **第二步：Creator 杠杆**(Month 3-6）签约 50-100 个抖音腰部创作者做冷启
3. **第三步：Remix 飞轮**(Month 6-12）一键 fork + 分成，creator:consumer 拉到 1:100
4. **第四步：平台化**(Month 12-24）和抖音 / 微信官方合作，变成 SDK 层

**第一步动作（Week 1-4)**:
- W1：拉齐团队（5 人，1 全栈 + 1 prompt 工程 + 1 游戏策划 + 1 增长 + 1 BD)
- W2：跑通最小 Pipeline（豆包 / Hunyuan pipeline API + HTML5 canvas 模板 10 个）
- W3：做 20 个 seed playable，灰度发布在小红书 / B 站
- W4：测 creator:consumer 初始比，决定下一步模板方向

**时间窗口**:⚠️ **12-18 个月（P0 紧迫）** — 2027 年前被 YouTube Playables / Roblox / 抖音官方吃掉的概率 > 60%

**多模块佐证**:
- 模块 2.3:Krafton × Inworld 深度绑定 · inZOI 2025.03 EA 商业化上线 【S-2.3-*】
- 模块 2.1：逆水寒 AI NPC 反面案例（G01 / G04) 【S-2.1-*】
- 模块 3 · T-1:middleware 投资主题 【S-3-T1】

**为什么确定性**:1) 东亚第一个跑通的 Runtime AI 商业案例；2) 大厂内部工具化惯性强，授权比自研快；3) 2026-2027 韩国 Lab 集中爆发时间窗 【S-2.3-*】

**TAM 估算**：全球 AAA 游戏 AI NPC 授权市场 **$200-500M**(2028)

**可复用路径**:1 款 showcase 游戏 → 技术服务收费 → 并购退出给大厂

**Week 1-4**：展示版 demo 立项 + BD 1 家大厂 PoC 意向

**时间窗口**：中期 18-30 个月

## §5. 探索性机会（4 个弱判断）

> **标签**:🧪 探索性假设 — 未被多模块数据完整验证，高回报但高风险

### E1. Telegram Mini App × AI(Catizen 路径）🧪
**服务视角**:A / B
- **假设**:Catizen / Hamster Kombat 证明 Telegram Mini App 的 IAA + Web3 经济学能跑，若叠加 Pipeline AI 生成内容，可复制 Mini App × AI 奇点
- **缺口**:Catizen 真实 CPM / ARPDAU 未公开（2.4 §8 待补）
- **第一步**：做 1 个 Mini App MVP 测 3 日留存
- **为何弱判断**:Telegram 生态是否稳定、中国团队出海是否能吃到 Web3 红利两大未知

### E2. 订阅制 AI 陪伴 × 教育壳 🧪
**服务视角**:A / D
- **假设**：把 AI 家教 / AI 口语和 Streak 游戏化深度融合，服务 K12 海外市场（避开国内教培监管）
- **第一步**：选 ESL 口语做 MVP，测订阅价格点
- **为何弱判断**：教育 K12 监管 + 付费意愿在不同国家差异巨大

### E3. 日本 VN × AI 破局（低概率高回报）🧪
**服务视角**:C / D
- **假设**：找 1 家日本中型 Galgame 厂（Marvelous / FuRyu / Visual Arts / Key）授权 IP 做 AI 化续作，用"粉丝续命"叙事破 VN 作者性锁死
- **缺口**：日本 Galgame 中型厂 AI 动作几乎零公开信息（2.4 §8 gap)
- **为何弱判断**：日本 VN 作者性 + IP 法务结构锁死**系统性阻止**(2.3 明确判断），成功率 < 10%，但一旦跑通是独门独路

### E4. 人格金标 eval SaaS 🧪
**服务视角**:B
- **假设**：模块 4 强烈建议 "人格金标 eval" 作为 Day 1 工程优先级，这是一个尚未出现独立玩家的细分
- **第一步**：做一个 eval 工具，免费给 3 家 AI 陪伴公司试用
- **为何弱判断**:Tooling 赛道的客户基数是否足够支撑独立 SaaS 不清楚

## §6. 风险警示（6 条 Don't)

| # | 不要做 | 反面证据 | 模块 |
|---|---|---|---|
| D1 | ❌ 不要做正面 "Runtime AI × IAA 小游戏" 交集 | 经济学真空，GPT-4o-mini $0.014/h vs ARPDAU < $0.05 | 1.1 · 1.3 · 3-G06 |

| D3 | ❌ 不要给 AAA 做内嵌 AI NPC（非授权合作） | 逆水寒 AI NPC 10 万条对话后玩家发现只会反刍设定 | 2.1 · 3-G01/G04 |
| D4 | ❌ 不要做 "AI washed" 只用 AI 做 PR 的项目 | 2024 Q4 死亡潮（Fable / Faulty / Vibecheck / Artie / Dreamworld) | 2.2 |
| D5 | ❌ 不要挑战日本 VN 作者性 + IP 法务 | Portopia AI Demo 翻车，日本 AAA 清一色内部工具化 | 2.3 |
| D6 | ❌ 不要在 AI 游戏里跑传统 A/B 测试 | Prompt 改动 / filter 更新会引发人格漂移 grief 退坑，传统 KPI 系统性误导 | 4.4 |
