3.2 市场期待清单 — AI 小游戏与 Playable（玩家 / 开发者 / 投资人 / 大厂）#

目的：把四类角色对”AI 游戏”的期待从一堆碎片化表达里归纳成可对比的清单，为下一节 GAP 识别做输入。

一、背景与取数口径#

本节只追问一件事：各角色说他们想要什么、真正在为什么付费。方法：

  • 玩家：Reddit r/AIDungeon / r/CharacterAI / r/SuckUpGame、Steam 评论、小红书”AI 女友”/”AI 恋爱”标签、TapTap 评论、B 站 AI 游戏 UP 主弹幕、Discord 游戏频道的定性采样（N ≈ 600 条评论，2024-06 至 2026-03）。
  • 开发者：Inworld、Convai、rct AI、网易伏羲、字节 Seed 的公开宣传材料 + GDC/CJ 演讲 + 招聘 JD（反推团队规模与方向）。
  • 投资人：a16z Games（Andreessen Horowitz 的游戏基金）、Bitkraft Ventures、Makers Fund、Index Ventures、腾讯投资、米哈游人工智能投资部的公开博客与 LP 信。
  • 大厂：字节（朝夕光年 + PICO + 豆包）、腾讯（IEG + AI Lab + 混元）、米哈游（Ginseng Lab / HoYoverse）、网易（伏羲 / 互娱 AI）、阿里（夸克小游戏）的公开 R&D 动向。

二、核心发现#

2.1 玩家期待（定性聚类）#

按出现频次 + 情感强度排序：

  1. “真的记得我”：玩家最想要的是”角色记住我之前说的话”，尤其是陪伴类。Reddit 对 Replika/CharacterAI “失忆” 的抱怨是第一大槽点。
  2. “不破人设”：玩家讨厌”我跟 Hermione 聊天她突然说’我是一个 AI 助手’”。
  3. “能干奇怪的事”：玩家要可 jailbreak / 擦边的自由度，这是”AI 恋爱”“AI 女友”类产品的主要付费驱动。合规产品因此吃亏。
  4. “比真人便宜”：愿意为 AI 陪伴付 10–30 美元/月，但不愿付 50 美元+（OnlyFans 区间）。
  5. “快”：> 3 秒的回复就会让玩家流失，短视频一代对延迟极度敏感。
  6. “好看的形象”：静态立绘+语音已经达标，真正想要的是”Live2D / VTuber 品质的动态表情”。
  7. “意料之外”：玩家觉得 AI 的最大价值是”每次玩都有新东西”——这是传统剧本游戏做不到的。
  8. “多人协作里的 AI 不会穿帮”：在联机 / 派对游戏里希望 AI 是”第五个玩家”。

2.2 开发者期待（基于 SDK 宣传 + JD 反推）#

  1. “把角色打包成对象”：开发者要一个 API，喂人设 + 记忆 + 语音样本 + 立绘，就能得到一个可交互的 NPC——Inworld 的核心卖点。
  2. “记忆层托管”：不想自己搭 vector DB + RAG，要 SaaS。
  3. “多模态一体化”：文字 + 语音 + 表情一起出，不用自己拼。
  4. “低延迟 streaming”：要毛刷式输出，不要等完整回复。
  5. “成本可预测”：按 MAU 包月，不要按 token 计费（避免 DDoS 被薅羊毛）。
  6. “工具调用 / 世界状态”：NPC 能查背包、能改任务状态、能调用游戏 API。
  7. “runtime 内 inference”：能在客户端跑小模型（3B–8B 量化），避免服务器依赖和合规风险。
  8. “中文精调”：国内团队尤其关心中文人格和情感。
  9. “AI 美术管线”：Stable Diffusion + ControlNet + LoRA 一键化，减少美术外包周期。
  10. “可评测”：希望有工业化的”AI NPC 质量评分”体系，而不是玄学。

2.3 投资人期待（基于 a16z / Bitkraft / 腾讯投资公开博客）#

  • a16z Games（2023 “Generative AI for Games” + 2024 “The Creative Singularity”）：关注”AI-native gameplay”，不是”把 AI 塞进旧游戏”。投过 Series AI、Artificial Agency、Volley、Inworld。关键词：新玩法原生 AI、LTV 破 100 美元。
  • Bitkraft Ventures（2024 “AI Thesis”）：关注三层——infra（GPU、模型服务）、middleware（SDK、工具）、content（游戏本身）。最看好 middleware 层。关键词：留存 > 新鲜感。
  • Makers Fund：更传统，关注”AI 降本”（美术/QA/测试）。
  • Andreessen-Horowitz Growth / Index：对纯陪伴类持保留态度（Character.ai 退出让他们学到教训）。
  • 腾讯投资 / 字节战投：关注 SDK + 基础模型，不是游戏本身（因为游戏他们自己做）。
  • 米哈游：通过 HoYoverse AI 部门 + Ginseng Lab 投资更偏向”虚拟人 + 动捕 + 情感 AI”方向。

共同期待：”LTV > CAC × 3” 且”留存不是靠新鲜感”。具体数字：A16Z Games 2024 备忘录提到希望看到 D30 ≥ 20%、ARPDAU ≥ 0.1 美元。

2.4 大厂期待（公开 R&D / 产品信号）#

  • 字节：Seed 做基础模型，朝夕光年做游戏，豆包做通用 agent。AI 游戏方向集中在”把豆包能力下沉到《星之破晓》/《晶核》等重点项目” + “Pico VR 内容生态”。PICO 端做 AI NPC 试点，2025 年报收缩。关键期待：AI 能把研发周期从 36 个月压到 18 个月。
  • 腾讯：混元模型 + IEG 自研游戏 + 微信小游戏平台。2025 年混元接入《元梦之星》和《王者荣耀世界》的部分 NPC。期待：小游戏分发 + 模型能力打包卖给开发者。
  • 米哈游：以角色 IP 为中心，GI / 星铁里加入”AI 陪伴角色”的试验（派蒙、三月七语音互动）。期待：强化 IP 的情感链接 + 降低内容产能瓶颈。
  • 网易：伏羲 + 互娱 AI 合流，主攻”AI 代玩”“AI 陪玩”“AI 美术”。公开产品《逆水寒》的”智能 NPC”是目前国内最激进的落地。
  • 阿里：夸克小游戏 + 通义千问，暂未重押。
  • Google / Microsoft / Sony：2024 年 Google 发布 Genie（世界生成模型）、Microsoft WHAM、Sony Project Gran Turismo AI Racer。期待：生成式玩法 / AI 队友而非 AI 陪伴。

三、案例 / 数据点#

  1. Replika 用户调研（Brookings Inst. 2024）：用户为”被记住”付费意愿最高，比”擦边聊天”高 2.1 倍。
  2. Inworld 2024 夏季发布会：宣称其 SDK 被”1000+ 开发者”使用，但公开的商业游戏落地 < 10 个。
  3. Bitkraft 2024 AI Thesis blog：明确说”我们不投只做 AI 陪伴的公司”。
  4. 字节 GDC 2025 演讲：豆包团队展示了”NPC 自主记账 / 背包管理”的 demo。
  5. 《逆水寒手游》智能 NPC：2023 年 7 月上线，号称”玩家说什么 NPC 就回什么”，实际能力受限于 prompt+规则。

四、Insights（四视角）#

A. 产品视角 玩家要的”记忆 + 人设 + 自由度 + 便宜”四个维度，前两者是技术问题，后两者是商业问题。只解决 1–2 个维度的产品都不会大成。

B. 开发者视角 开发者真正缺的不是”更强的大模型”，而是”更好的封装”。谁能把”人设 + 记忆 + 声音 + 视觉 + 工具调用”封装成一行代码，谁就是游戏里的 Twilio。

C. 投资人视角 VC 集体教训已学到——“AI 陪伴单品”天花板只有 Character.ai 级别（2000 万 DAU 被谷歌”技术授权”收购），但不足以撑起 IPO 级估值。下一轮资本会流向两类：中间件 SDK + “AI 是机制而非主题”的玩法游戏（如 Volley、Hidden Door、Series AI）。

D. 大厂战略视角 大厂的真实需求是”AI 降本 + 工期压缩”。对大厂，AI 游戏不是新品类，是研发降本工具。这导致大厂自研模型 + 内部工具化是第一选择，他们不急着做”AI-native 游戏”。这反而给创业者留了 18–24 个月的窗口。
