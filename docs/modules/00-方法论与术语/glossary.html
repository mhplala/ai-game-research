术语表 — glossary.md#

AI 游戏 / 小游戏 / Playable 体验赛道的核心术语。研究过程中持续补充。

AI / 技术#

术语 英文 定义
GenAI Generative AI 生成式 AI，包括文本 / 图像 / 音频 / 视频生成
LLM Large Language Model 大语言模型（GPT-4 / Claude / Gemini / Llama 等）
LLM Agent — 由 LLM 驱动的自主 agent，可调用工具 / 记忆 / 决策
Agentic Gameplay — AI agent 作为玩法核心机制的游戏（非 scripted NPC)
NPC Non-Player Character 非玩家角色，AI NPC 指 LLM 驱动的对话 / 行为动态的角色
RAG Retrieval-Augmented Generation 检索增强生成，让 LLM 有”记忆” / 世界观上下文
Character Consistency — AI 角色在长对话 / 多轮交互中保持人格一致性
Token Economics — 基于 LLM token 计费的成本结构 / 利润模型
PCG Procedural Content Generation 程序化内容生成，传统方法 + AI 增强
Fine-tuning — 针对特定任务 / 角色 / 世界观微调基础模型
Diffusion — 扩散模型，用于生成图像 / 贴图 / 角色立绘
WebGPU — 浏览器内 GPU 通用计算 API，小游戏高性能渲染

游戏 / 商业#

术语 英文 定义
小游戏 Light Game / Instant Game <50MB 即开即玩的轻量级游戏，主要在微信 / 抖音 / Meta 等平台
Playable Ads — 可互动的广告形式，用户可实际操作产品体验
互动 H5 — 基于 HTML5 的互动内容，常用于营销 / 品牌推广
H5 游戏 — 基于 HTML5 标准的轻量游戏，可在浏览器 / 移动端直接运行
PWA Progressive Web App 渐进式 Web 应用，可离线运行的 web-first 应用
GaaS Games as a Service 游戏即服务，持续运营 + 内容更新的游戏商业模式
IAP In-App Purchase 应用内购买 / 内购
Gacha — 抽卡机制（日式扭蛋），主要变现手段
Live Ops — 持续运营（活动 / 更新 / 商品），维持玩家留存
D1 / D7 / D30 — 次日 / 7 日 / 30 日留存率
CAC Customer Acquisition Cost 用户获取成本
LTV Lifetime Value 用户生命周期价值
ARPPU Average Revenue Per Paying User 每付费用户平均收入
ARPDAU Average Revenue Per Daily Active User 每日活用户平均收入
CPM Cost Per Mille 每千次展示成本（广告）
DAU / MAU Daily / Monthly Active Users 日活 / 月活

玩法范式#

术语 定义
Text Adventure 文本冒险（AI Dungeon 范式）
Visual Novel / VN 视觉小说，AI VN 指 LLM 动态生成剧情的视觉小说（1001 Nights 等）
Sandbox 沙盒游戏，AI 沙盒代表为 Altera 的智能 NPC 社区
Dating Sim 恋爱模拟，AI 恋爱指 Character.AI / Replika 方向
Role Play / RP 角色扮演，AI RP 指用户与 AI 扮演不同角色互动
Roguelike 每局随机生成的地下城游戏，AI 增强 PCG
Idle / Incremental 挂机 / 放置游戏，小游戏主流品类（《咸鱼之王》《寻道大千》）
Match-3 三消（《羊了个羊》式）
Simulation 模拟经营，AI 模拟可能应用于 NPC 行为
