2.4 跨区域趋势综合 — 全球 AI × 小游戏 × Playable 三大命题#

编写：主 agent 整合 输入:2.1 中国市场（27 案例）· 2.2 美国市场（32 案例）· 2.3 日韩市场（27 案例）· 模块 1/3/4/5 已完成产出 最后更新:2026-04-13 定位：本文件是跨地域的趋势级综合，按”趋势轴”而非”地域轴”组织，作为模块 06 机会地图的直接输入。

0. TL;DR#

全球 AI 游戏与小游戏赛道在 2022-2026 形成了四大区域画像 + 三大核心命题。

四大区域画像#

市场 角色 关键特征
美国 主战场，退出模式已浮现 middleware 层跑出（Inworld ~$500M),reverse acqui-hire 闭环（c.ai → Google / Inflection → MS),C 端独立 IPO 路径关闭，资本 2024H2 全面转向 B2B
中国 主战场，但”游戏性”被版号扭曲 星野 / 猫箱 / 筑梦岛 全部藏在陪伴品类，量子绘梦 Anuttacon 团队注册美国，”游戏”标签成监管负债
日本 应该大却哑火 VN 土壤被作者性 + IP 法务锁死，VTuber 生态挤占 AI 陪伴位置，AAA 只做内部工具，Portopia AI Demo 翻车让日式 LLM 游戏信心受挫
韩国 雷声大雨将落 4 大 Lab(MOGIA / VARCO / Nexon Intelligence Labs / Smilegate AI)+ 自研 LLM 全到位，2026-2027 才是爆款窗口（inZOI + Zeta 是仅有的两个商业验证）

三大核心命题（贯穿全球）#

  1. 经济学真空带 — AI × 小游戏的交集现在基本不存在，Token/session $0.014(GPT-4o-mini)vs IAA ARPDAU < $0.05 吃不起 AI 税。唯一解是端侧小模型 或 平台补贴（字节豆包 / 腾讯 Hunyuan 正在做）。
  2. 游戏性定义权脱离 — 最优形态不是把产品叫”游戏”，而是把游戏内核藏进陪伴 / 工具 / 教育 / 小说 / 广告的壳。中国版号强制 + 美国自发选择 + 日韩文化偏好，三条路径都指向同一结论。
  3. 退出模式分化 — 美国 reverse acqui-hire(c.ai $2.7B / Inflection $650M)/ 中国巨头绑定内生 / 韩国延后爆发 / 日本基本不退出。创业生态密度和这四条模式的闭合性正相关。

中国跟进度总评#

  • T1(AI 陪伴）:✅ 全球领先（星野 3 亿流水）
  • T5（平台补贴）:✅ 全球独有（豆包补贴是破真空带的唯一路径）
  • T6(AI 买量）:✅ 运营侧最成熟
  • T3（大厂实验）:⚠️ 有 Lab 但没出货
  • T2(AI-native 独立）:❌ 几乎空白，Anuttacon 跑美国
  • T4(middleware):❌ 几乎空白，rct AI 转型未成
  • T7(AI 生成即玩法）:❌ 少
  • T8(Playable × AI):⚠️ 广告侧有但没产品化
  • T9(AAA 卖点）:❌ 逆水寒反面案例打击信心

总结：中国领先的是”规模化运营”维度（T1/T5/T6），落后的是”技术创新 / 中间层”维度（T2/T4/T7）。这和 Phase 1 模块 3 的判断完全一致：G01 长时记忆 + G06 Token 倒挂 + G10 LTV 天花板 三堵墙是全球性的，而中国的额外障碍是监管 + middleware 空白。

2. 区域战略画像（完整版）#

2.1 美国市场：“理念驱动 + B2B middleware + 顶部 acqui-hire”#

核心打法:AI-native 创业潮 → VC 密集投资 → 一部分跑通（付费 Steam / 订阅 C 端）→ 大部分失败 → 2024H2 资本转向 B2B middleware 和 acqui-hire 退出。

创新节奏: - 2022-2023 早期热:AI Dungeon / Character.AI / Latitude / Inworld / Hidden Door / Altera 密集融资 - 2024H1 分化:Suck Up! / 1001 Nights / Infinite Craft 跑出独立成功，但 C 端公司（Fable / Vibecheck / Artie / Faulty）死亡潮 - 2024H2 退出潮:Character.AI $2.7B / Inflection $650M，两个 reverse acqui-hire 定义 C 端陪伴天花板 - 2025 middleware 年:Inworld / Rho Series AI / Convai / Rosebud 成为资本新宠

最值得中国借鉴的 3 个案例: 1. Suck Up! — < 5 人 solo 团队 / Steam 爆款 / $9.99 买断 — 证明”小而美 AI-native“是现阶段唯一跑通的独立游戏形态 2. Infinite Craft(Neal Agarwal）— zero funding solo / Poki 数千万玩家 / 证明极简玩法 + 低频 LLM 调用 + 广告变现可以跑通，是打破 G06 Token 倒挂的唯一已验证路径 3. Inworld AI — B2B middleware 估值 $500M — 证明”卖铲子给淘金者“比”自己淘金”更稳，是 T-1 投资主题的现实版

2.2 中国市场：“陪伴品类壳 + 平台补贴 + 运营侧 AI”#

核心打法: - 真 AI-native 藏在陪伴品类：星野 / 猫箱 / 筑梦岛 - 游戏品类的 AI 只用于运营：寻道大千 / 咸鱼之王（买量素材） - 大厂通过 Lab 孵化 + Token 补贴构建生态：腾讯 Hunyuan Game / 字节豆包 / 抖音嘘嘘岛 / 快手可灵

代表品牌与数字: - 星野（MiniMax):2024 流水 > 3 亿，付费率 3-5%，头部月付 ¥1000+ - 猫箱（字节）:DAU 破百万，战略归属在”豆包家族” - 筑梦岛（美团投）：AI 角色扮演新锐 - Whispers from the Star(Anuttacon / 蔡浩宇）：中国第一个挂”游戏”名的 AI-native，但团队注册美国规避版号 - 寻道大千（三七互娱）：微信 Top 1,AI 仅做买量 - 逆水寒 AI NPC（网易）：AAA 的反面案例，玩家 10 万对话发现 NPC 反刍设定

独特路径: - 豆包免费 tokens 是全球独有的”平台补贴破真空带“策略，可能是唯一能打通 AI × 小游戏交集的力量 - 版号扭曲 是双刃剑：阻止 AI-native 挂游戏名，但也逼出”陪伴品类壳”创新

最大结构性缺失: - 没有 middleware 层（仅 rct AI 在转型） - 没有独立退出路径（头部绑定巨头内生，没人能独立 IPO)

2.3 日本市场：“VN 沃土 + AAA 内部工具 + VTuber 生态挤占”#

本应该是全球 AI-native 游戏最好的土壤: - VN（视觉小说）文化 × AI 动态叙事 = 天作之合 - 同人创作生态 + Galgame 细分市场 + 成熟付费心智

但实际哑火的三个原因: 1. 作者性与 IP 法务：日本 VN 市场由少数老牌作者主导（Key / Nitroplus / Visual Arts），对 AI 生成内容有严重的”作者性“抵触，加上 IP 法务保守 2. VTuber 生态挤占:hololive / にじさんじ 等 VTuber 平台已经占据了”虚拟角色 × 情感投入”的位置，Cotomo / Rinna 这类 AI 陪伴根本竞争不过真人扮演的 VTuber 3. AAA 大厂清一色”内部工具化”:Square Enix AI Division / Capcom CEDEC / Bandai Namco / Konami 全部不对外营销，唯一公开翻车的 Portopia AI Demo 反而让业界对日式 LLM 游戏信心受挫

结论：日本不是”雷声大”，是”应该大却哑火“。2026-2030 也难有大变化，结构性锁死。

2.4 韩国市场：“4 大 Lab 备战 + 底座自研 + 2026-2027 爆发窗口”#

Krafton MOGIA(2024.06 成立）是东亚唯一把 AI NPC 作为”用户可见卖点“商业化上线的大厂（inZOI “Smart Zoi” 2025.03 EA）。底层与 Inworld 深度绑定（Krafton 是 Inworld 战略投资人）。

其他 3 个 Lab: - Nexon Intelligence Labs(intelligencelabs.nexon.com) - NCSoft VARCO LLM(2023.08 发布）+ Project G - Smilegate FutureLab AI

韩国 AI 陪伴: - ScatterLab Zeta(2024)MAU 数百万 - Character.AI 韩国为 Top 5 流量国 - 被 K-drama IP + 语言封闭市场托举

为什么”雷声大雨将落”: - 4 大 Lab + 底座 LLM 自研全到位 - 但 2026 前仅 inZOI + Zeta 两个商业验证 - 2026-2027 才是真爆款窗口 - 值得 Track 但不是当前下注窗口

3. 创新维度十字象限分析#

横轴:AI 密度（高：AI 是核心机制不可移除 vs 低：AI 只是辅助） 纵轴:商业化成熟度（高：已验证 LTV/ARPDAU 足以覆盖成本 vs 低：烧钱实验）

四象限定位#

象限 定义 代表 区域
Q1 高 AI × 高成熟(已跑通) AI-native 且商业化验证 Character.AI(C 端） / Replika / Suck Up!(Steam 付费）/ 星野（订阅）/ ScatterLab Zeta 🇺🇸🇨🇳🇰🇷
Q2 高 AI × 低成熟(烧钱实验) AI-native 但 LTV 不够 / 成本高 AI Dungeon / Latitude / Hidden Door / Altera / Fable（死）/ Dreamworld（死） / Anuttacon Whispers 🇺🇸🇨🇳
Q3 低 AI × 高成熟(AI-washed 运营) AI 只辅助，核心玩法传统 寻道大千 / 咸鱼之王 / 逆水寒 AI NPC / 日本 AAA 内部工具 / 腾讯 Hunyuan Game 当前状态 🇨🇳🇯🇵
Q4 低 AI × 低成熟(被淘汰带) AI 只是 PR 噱头，商业化也不行 大量 2023-2024 失败的 AI 游戏创业公司 全球

三个关键判断#

  1. Q1 是唯一真实的赛道，但它不是”AI × 小游戏”的交集 — 它是 “AI × 付费订阅 / Steam 买断 / 陪伴应用”。这和 brief 的原始命题有关键偏移：我们要研究的交集现在是空的，真正有价值的是它的”邻域“。
  2. Q2 是最大的陷阱 — VC 2022-2023 都在这里下注，但 2024H2 退出潮证明了 LTV 天花板，2025 开始向 Q3 / middleware / 端侧模型转移。
  3. Q3 是中国小游戏的现状 — 用 AI 做运营（买量素材）而非玩法。这不是未来，但是现阶段的主流收入。

对创业者的战略建议: - 首选:玩法/体验驱动 + 小而美团队（对标 Suck Up! / Infinite Craft）— 风险最小，门槛最低 - 次选:成分/技术驱动 middleware — 需要 B2B 销售能力，但有 Inworld 范本可抄（T-1 投资主题的执行版） - 避免:情感/IP 驱动纯 C 端独立公司 — LTV 天花板已被 Character.AI 证明，$20-40 死线 - 中国特殊:情感驱动但藏在陪伴壳 + 绑定巨头获 Token 补贴（星野 / 猫箱模式），这是唯一跑通的中国路径

5. 迁移路径判断 — “AI × 小游戏” 真空带如何被打通#

5.1 Scenario A：平台补贴路径（中国主导）#

逻辑：字节豆包 / 腾讯 Hunyuan / 快手可灵把 Token 成本归零给小游戏开发者，人为破真空带。

已看到的信号: - 豆包免费 tokens 对接小游戏开发者 - 腾讯 Hunyuan Game 提供游戏场景优化模型 - 抖音嘘嘘岛 PICO 沙盒

2-3 年时间表: - 2025 Q3：微信小游戏 / 抖音小游戏出现第一批”免费 AI NPC”试验（但仍在买量侧） - 2026 Q1：第一个非买量类的”AI × 小游戏”跑出 ¥1 亿流水 - 2027：中国小游戏平台出现 AI-native 品类

瓶颈：平台补贴只是延后 Token 成本，不是消除。如果平台策略变（豆包涨价 / Hunyuan 收缩），整个生态崩塌。

5.2 Scenario B：端侧小模型路径（全球主导）#

逻辑:Llama 3B / Qwen 3B / Phi-4-mini 部署到手机端，推理成本归零。

已看到的信号: - Apple Intelligence(M 系列 Mac / iPhone 16 Pro+) - Google Gemini Nano on Android - Qwen 2.5-3B / Llama 3.2-3B / Phi-4-mini 开源

2-3 年时间表: - 2025:Web LLM(WebGPU）可以在浏览器里跑 3B 模型，延迟 ~100 tokens/sec - 2026 H2：第一个端侧 AI-native 小游戏（浏览器内运行，零后端成本）出现 - 2027：微信小游戏 / Poki / CrazyGames 出现 Web LLM 类目

瓶颈：端侧模型能力有限，做 NPC 对话合格但做复杂 agent 不够；移动端电池/发热问题。

5.3 Scenario C:Playable Ads 壳路径（最被低估）#

逻辑：不做小游戏，做”播后体验” playable ads，把 AI 玩法嵌入广告本身。广告主买单，玩家零成本。

已看到的信号: - Catizen / Hamster Kombat(Telegram 小游戏，游戏 = 广告 = 代币） - Luna Labs AI playable 广告 - Telegram Mini App 的快速增长

机会点: - 广告主 ARPDAU = 几十美元（游戏广告 CPM $10-50），远高于 IAA $0.05 - AI 成本被广告预算覆盖 - 完全破真空带

瓶颈:Telegram / Kakao / LINE 等生态封闭，中美各自监管，规模受限。

5.4 Scenario D:”游戏壳 ≠ 游戏” 路径（最激进）#

5.5 Scenario E:UGC AI Playable Maker 路径(2025 新物种，本研究中期补入）#

逻辑：不做”AI 在游戏里实时推理”，而是做 “用户用 AI 在创作时生成 playable，玩家消费静态生成结果”。AI 只在 Pipeline 端被调用一次，玩家玩 100 次不用再调，彻底绕过 Token 经济学真空带。

为什么是独立路径: - Runtime AI 成本 = 0（生成完就静态化） - 分发成本 = 0(TikTok 式 feed 飞轮 + 一键 remix) - 玩家 = 创作者(UGC 社区，CAC 近零） - 这是”AI × 小游戏”真空带被打穿的最接近证明

已看到的信号（2025): - Rezona(MWM.ai，法国）— iOS + Android,”最离谱的 meme 游戏平台”,TikTok 式 feed + 一键 remix - LoopIt(MWM.ai，法国）— iOS 独家，”Make playables”,ASMR / 触感互动 / 小玩具 - YouTube Playables Builder(Google 2024-2025）— 平台级降维打击，让 YouTube creator 直接给订阅者做游戏 - Upit / Ludo.ai / Jabali.ai / Rosebud AI / Replit AI Game Builder / Websim — 相邻玩家矩阵 - 前辈参照:Roblox（前 AI 时代 UGC)/ Dreams(PS 商业失败）

2-3 年时间表（推断）： - 2025 H2:Rezona / LoopIt / YouTube Playables Builder 验证可行性，MAU 达到百万级 - 2026 Q1-Q2:TikTok / 抖音 / 快手 跟进，国内出现本土版 - 2026 Q3-Q4：字节 / 腾讯平台级入场（基于豆包 / 混元 Token 补贴 + 已有的短视频分发） - 2027：成为”AI 时代的 Roblox”候选者之一

瓶颈: - 内容生产的质量天花板:AI 生成 meme 游戏玩法简单，深度有限，长期留存能否维持？ - 商业化不清楚：目前看都还没赚钱，广告 / 订阅 / 付费 remix 哪条能跑通未知 - Dreams 前车之鉴:Media Molecule 的 Dreams(PS4/PS5）证明了 UGC 游戏创作器的长期商业化失败(UGC 创作者热情高，但普通玩家留存低） - 版权 / IP 合规：用户生成 meme 游戏极易卷入版权，Rezona 的 “unhinged” 定位是反面风险

为什么这是 “AI × 小游戏真空带” 的正确答案候选: - Scenario A（平台 Token 补贴）= 延后成本，不是消除 - Scenario B（端侧小模型）= 技术上还在早期 - Scenario C(Playable Ads 广告壳）= 被 Telegram / Catizen 卡在封闭生态 - Scenario D（游戏壳 ≠ 游戏）= 绕监管但不绕经济学 - Scenario E(Pipeline AI + UGC Feed)= 从根本上消除了运行时 AI 成本，且分发成本趋零 — 这是五条路径里最彻底的解**

6. 总结（Summary)#

  1. 全球 AI × 小游戏赛道 2022-2026 经历了三阶段：热潮（2022-2023)→ 分化（2024H1)→ 退出潮 + middleware 转向（2024H2-2025）。
  2. 四大区域画像：美国主战场已形成退出闭环；中国主战场但被版号扭曲；日本哑火；韩国延后爆发。
  3. 真正的”AI × 小游戏”交集现在不存在（经济学真空带），但它的四个邻域有跑通的案例：Steam/iOS 付费 AI-native(Suck Up!)/ C 端订阅 AI 陪伴（Character.AI / 星野）/ B2B middleware(Inworld)/ 广告运营 AI（寻道大千）。
  4. 三大命题跨区域一致：经济学真空带 / 游戏性定义权脱离 / 退出模式分化。
  5. Q1 象限是唯一真实赛道（高 AI × 高成熟），但它不在”小游戏 IAA”的价格带上。
  6. 四条迁移路径都在跑但各有瓶颈：平台补贴（中国）/ 端侧小模型（全球）/ Playable Ads 壳（Telegram 先行）/ 游戏壳 ≠ 游戏（中国头部已在做）。
  7. 韩国 2026-2027 是值得 track 的爆发窗口，中美 2024-2026 是当前主战场，日本结构性锁死难改。

7. 洞察（Insights）— 四视角交叉#

A. 产品经理 / 创业者视角#

  1. 不要做”AI × 小游戏”的正面交集 — 经济学不成立。做四个邻域之一。
  2. 首选 “AI-native 小而美 Steam/Poki 付费 + Indie” — Suck Up! / Infinite Craft 范式，门槛最低，最多 $100K 启动资金，3 人团队可做。
  3. 次选 “游戏壳陪伴 / 工具 / 教育” — 中国的星野模式，但需要内容运营能力。
  4. 避免 “情感驱动纯 C 端独立公司” — LTV $20-40 天花板已被证明。

B. 投资人视角#

  1. middleware 是 2025-2027 最大投资主题(T-1），中国市场空白尤其大 — Inworld 中国版是一个值得孵化的 thesis。
  2. 端侧小模型 + 游戏 infra 是第二大主题 — Web LLM / WebGPU / Apple Intelligence SDK 游戏化都在开局。
  3. 避免”纯 C 端 AI 陪伴独立上市”论（被 Character.AI / Inflection 证伪）— 除非你能接受 reverse acqui-hire 作为退出。
  4. 韩国大厂 Lab 是延后订单 — 2026-2027 爆发窗口，现在投韩国 B2B 中间层可能是最佳时点。

C. 行业观察视角#

  1. 资本 2024H2 全面转向 B2B middleware 是赛道成熟信号，意味着 “淘金者 → 卖铲子” 的经典范式切换发生了。
  2. 中国的”游戏性定义权脱离 + 版号强制扭曲” 是全球独特的行业结构，值得长期 tracking。
  3. “AI 税 15-40% ARPDAU” 的真空带是这一轮赛道的核心瓶颈，任何声称”AI 小游戏”的项目都需要明确说明他们怎么解决这件事。
  4. Krafton MOGIA + inZOI Smart Zoi 是东亚唯一大厂跑通”用户可见 AI NPC”的案例，值得作为范本研究（即使它是重度游戏不是小游戏）。

D. 战略规划视角（大厂 / 平台 / 战投）#

  1. 中国平台方（腾讯 / 字节 / 快手）的 Token 补贴战 是 AI × 小游戏真空带最可能的破局路径，战略上应持续补贴 — 和当年”抖音补贴短视频 MCN”同构。
  2. 大厂应该投资 middleware 而不是自己做游戏 — 腾讯 Hunyuan Game 目前属于 Q3 象限（AI 噱头），战略位置应向 Inworld 式平台转型。
  3. 端侧小模型 + WebGPU 联合生态 是平台方未被充分重视的机会 — 字节可以用 PICO / 抖音 Web 组合，腾讯可以用微信 WebView + 混元端侧组合。
  4. “游戏壳 ≠ 游戏” 战略 — 大厂应该鼓励旗下陪伴 / 工具 / 教育产品的游戏化，而不是拉一个 “AI 游戏” 独立品牌（独立品牌会触发游戏监管，融合进现有品牌反而规避了）。
  5. 韩国 Lab 合作窗口 是未被中美大厂充分利用的机会 — 和 Krafton MOGIA / Nexon IL 的技术授权 / 投资合作可能带来早鸟优势。

8. Gaps & 待核实#

  1. 端侧小模型在游戏中的实际性能 没有公开 benchmark（特别是 WebGPU + 3B 模型组合的首 token 延迟和吞吐）
  2. 豆包 / Hunyuan / 可灵对小游戏开发者的补贴具体金额 没有公开数字
  3. Krafton MOGIA 真实团队规模 / 投入预算 只有 PR 口径，需 DART 公告补
  4. Telegram Mini App 经济学 (Catizen / Hamster Kombat）的真实 CPM / ARPDAU 未公开
  5. Character.AI → Google $2.7B 授权后 a16z 等投资人的实际回报倍数 未披露
  6. Apple Intelligence SDK 对游戏开发者的具体开放时间表 未公开
  7. 日本 Galgame 中型厂商 AI 动作几乎零信息(Marvelous / FuRyu / Visual Arts / Key 等）

9. 对机会地图（模块 06）的直接输入清单#

A. 确定性机会（强判断，多模块佐证）#

  1. 端侧小模型 × 小游戏 / WebGPU 游戏 infra — T1 + T5 + G06 的三重交集

  2. 端侧 LLM + WebGPU infra:2025-2027 爆发期
  3. Playable Ads × AI 生成工具链：广告预算做护城河

C. 战略路径（大厂）#

D. 不要做什么（风险警示）#

  3. ❌ 不要做大 AAA 内嵌 AI NPC（逆水寒反面案例）
  4. ❌ 不要做”AI washed” 只用 AI 做 PR 的项目（Q4 被淘汰带）
