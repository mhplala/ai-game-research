1.4 玩家全景图#

模块 1.4 · AI 小游戏与 Playable 体验赛道 · 全球主要玩家结构化扫描

覆盖 85 个产品 / 品牌 / 机构，来自模块 2.1（中国 27)+ 2.2（美国 32)+ 2.3（日韩 27)+ 2.5(UGC AI Playable Maker，占位）+ 平台/监管补行。每条都填”双刀”分类（模块 1.1 §6）。素材 100% 复用现有案例，未新增调研。

TL;DR#

  1. 赛道高度集中在”Runtime AI-native × 陪伴/剧情”一个象限(25+ 产品），其中 Character.AI / 星野 / Zeta / 猫箱 已跑通”DAU 百万 + 流水亿级”，但大多数同类产品在Token 经济学真空带内挣扎。
  2. Pipeline AI for Creator 象限基本空白,2025 才出现 Rezona / LoopIt / YouTube Playables Builder 等 6-8 个玩家，是未来 2 年真正的破局路径（详见模块 2.5）。
  3. middleware 层（Inworld / Convai / Artificial Agency / Series）集中度高但付费客户少,2025 Q1 前后出现第一波缩编与并购信号（Character.AI 准退出、Hidden Door 缩编）。
  4. 中美日韩节奏差 6-12 个月：美国在 middleware 和 Runtime AI-native 已完成一轮洗牌，中国在 AI 陪伴侧刚刚跑通头部（星野/猫箱），日本仍停留在 AAA 内部管线，韩国出现 Krafton MOGIA 这种东亚唯一跑通 AAA Runtime AI 的案例。
  5. AI-washed 象限是小游戏主战场（寻道大千 / 咸鱼之王 / 无尽冬日），流水最高但 AI 可移除，不是真正的”AI 游戏”。

三、重点品牌 10 案例展开#

① Character.AI(S-2.2-B1）— AI 陪伴赛道定义者 / 顶部退出案例#

Character.AI 是整个 Runtime AI-native 赛道的定义者。由前 Google LaMDA 核心成员 Noam Shazeer 与 Daniel De Freitas 于 2021 创办，2023 峰值 MAU 2000 万+，用户日均对话时长一度超过 TikTok,C 轮估值 $10B。但 2024 Q3 Token 成本占比 > 65% 导致订阅 ARPU 无法覆盖推理，最终以 $2.7B 非股权”授权费”形式将核心团队卖回 Google，产品本身留给剩余团队维持。这是赛道最重要的警示：即使做到 MAU 2000 万 + 订阅 5% 渗透，Runtime AI 在 C 端仍然跑不通。它直接定义了”Token 经济学真空带”的上边界。刀 1 = native · 刀 2 = Runtime · 评分 5。

② Inworld AI(S-2.2-C1）— middleware 领跑者#

Inworld 是 middleware 阵营唯一拿到 $120M+ 累计融资、估值约 $500M 的玩家，投资方含 Lightspeed / a16z / Disney / Intel。它做 NPC 行为 + 对话 SDK，绑定 Ubisoft NEO NPC、NVIDIA ACE、Krafton PUBG Ally 三个重量级 demo。商业模型是 per-session token 分成 + B2B 订阅。但 2024 末 Inworld 实际 B 端付费客户数仍在个位数到十几家，Ubisoft NEO NPC demo 后 Ubisoft 并未真正上线。它是”技术验证完成但 B 端需求未启动”的典型。刀 1 = enabler · 刀 2 = P-Dev · 评分 5。

③ 星野 / MiniMax(S-2.1-02）— 中国 AI 陪伴唯一跑通#

星野是中国 Runtime AI-native 赛道唯一真正跑通营收的产品。MiniMax 把前身 Glow 被下架后重启，走”乙女+角色扮演+语音”路线，2024 年流水 > 3 亿元，2025 Q1 跨过 5 亿元，是中国 AI 陪伴头部。海外双胞胎 Talkie 在美国跑到 Top 10 社交应用。关键差异：星野同时在 runtime 塞了三层内容（预置人设 + UGC 角色 + 玩家小说剧情），通过降低生成次数 + 集中在乙女高付费品类，把 Token 成本占比压到 30% 以下。刀 1 = native · 刀 2 = Runtime · 评分 5。

④ Suck Up!(S-2.2-A2）— 小而美 AI-native 范式#

Proxima Enterprises 开发的 Suck Up! 是 Steam 上第一款真正意义上的 LLM-native 游戏。玩法是扮演吸血鬼说服 NPC 邀请你进屋，完全依赖 LLM 理解玩家自由输入。自筹资金开发，2023 抢先体验以 $14.99 买断发售，Steam 累计 5 万+ 好评、约 $1M+ 营收。它的意义不在规模而在跑通商业模型：买断制 + 玩家一次性付费 + 游戏生命周期内 Token 成本可控（平均每玩家 $0.5-$1）。是”buy-to-play 包住 Runtime 成本”的唯一成功范式。刀 1 = native · 刀 2 = Runtime · 评分 4。

⑤ AI Dungeon / Latitude(S-2.2-A1）— GenAI 游戏鼻祖 / 警示案例#

2019 年推出的 AI Dungeon 是全球第一个 LLM 游戏，最初基于 GPT-2，后迁 GPT-3。峰值付费用户 10 万+，但 2021 年因 NSFW 内容遭 OpenAI 收紧政策导致一次用户流失；此后陷入”自研模型性能差、用 GPT 成本撑不住”的两难。2024 年公司 Latitude 仅剩十几人，产品靠核心付费用户维持。它的警示价值：第一个不一定赢，依赖第三方 API 的 runtime 产品永远受上游政策制约。刀 1 = native · 刀 2 = Runtime · 评分 3。

⑥ Krafton MOGIA + inZOI Smart Zoi(S-2.3-14/15）— 东亚大厂唯一跑通 AAA Runtime AI#

韩国 Krafton 于 2023 底成立 MOGIA(Mobile/Open-world Generative Intelligence Agent）工作室，30+ 人，专做 Runtime AI 在 AAA 中的落地。首个成果 inZOI 的”Smart Zoi”是一套本地化 LLM 小模型 + 云端大模型分层的 NPC 行为系统，2025 年 3 月 Early Access 发售，Steam 预约 Top 10。这是东亚大厂中唯一一个真实发售、真正 Runtime 依赖 AI 的 AAA 产品，比 Ubisoft NEO NPC demo 和 Square Enix Portopia demo 都更进一步。与 NVIDIA + Inworld 合作的 PUBG Ally 是第二个 demo 试点。刀 1 = native · 刀 2 = Runtime · 评分 5。

⑦ Infinite Craft(S-2.2-A6）— solo 零融资现象级#

Neal Agarwal 一人 2024 年 1 月上线的网页游戏：玩家把两个元素拖到一起，后端调用 LLM 生成新元素名。发布 7 天 DAU 破 100 万，现象级。关键是它的 AI 用法创新：用 LLM 生成但缓存结果，相同组合只算一次，之后全球用户共享结果库，Token 成本摊到接近零。这是 Runtime + Pipeline Hybrid 的教科书案例，也是”单人 solo 零融资 vs Character.AI $10B 融资”的鲜明对照。刀 1 = native · 刀 2 = Hybrid · 评分 4。

⑧ 寻道大千（S-2.1-16）— 中国小游戏 AI-washed 标杆#

三七互娱 2023 年上线的微信/抖音放置卡牌小游戏。2024 年日流水长期 Top 1，累计流水 40 亿+。它和 AI 的关系：核心玩法 100% 非 AI（放置卡牌是 2015 年技术），但买量素材 70%+ 由 AI 生成,AI 生成视频/图片投放 CPA 比人工素材低 20-30%。这是”AI-washed × Pipeline-Dev”的完美样本：游戏本身可完全移除 AI，但 AI 在素材层让它比对手多赚 30% 利润。刀 1 = washed · 刀 2 = P-Dev · 评分 4。

⑨ Hidden Door(S-2.2-A4）— a16z $20M 后 2025 Q1 缩编警示#

Hidden Door 由前 Oculus Story Studio 创始人 Hilary Mason 创办，愿景是”把你最爱的书变成 RPG”,a16z SPEEDRUN 领投 $20M Series A(2023）。技术是自研小模型 + 授权 IP。但 2024 闭测反馈平平，2025 Q1 坊间传出缩编至 10 人以内。警示点：拿到顶级 VC 头部票也不够，在 Runtime AI 真空带内，没有跑通的产品市场契合。刀 1 = native · 刀 2 = Runtime · 评分 2。

⑩ Whispers from the Star / Anuttacon(S-2.1-11）— 蔡浩宇首款挂游戏名的 AI-native#

Anuttacon 是米哈游前 CEO 蔡浩宇 2024 年在美国创立的个人项目。首作 Whispers from the Star 是一款”玩家通过文字/语音和一个被困外星的角色实时对话，帮她生存”的 AI-native 游戏，TestFlight 2025 年初开放。这是中国创作者在美国做 AI-native 游戏第一案，也是”米哈游体系离开后下一步”的风向标。规模未知，但它的战略意义巨大：米系人物亲自下场验证”Runtime AI 游戏能否做成一个可售卖的产品”。刀 1 = native · 刀 2 = Runtime · 评分 4。

4.3 2024-2026 赛道演变 5 个关键节点#

  1. 2024 Q2 — AI Dungeon/Replika 增长见顶，首次暴露”订阅 ARPU 无法覆盖 Token”的全行业问题。
  2. 2024 Q3 — Character.AI → Google”授权”退出,$2.7B “非收购收购”，整个 Runtime AI-native 赛道被迫重新估值。
  3. 2024 Q4 — 星野/Talkie 突破 3 亿流水，证明东亚市场 AI 陪伴可以在”乙女 + 高付费”细分下跑通。
  4. 2025 Q1 — Krafton inZOI Smart Zoi EA 发售 + Hidden Door 传缩编,AAA Runtime AI 首次实测成功 vs 中型创业公司失败，赛道向两端分化。
  5. 2025 Q3-Q4 — MWM Rezona/LoopIt + YouTube Playables Builder 上线,Pipeline-Creator 新物种首次集体浮出水面，模块 1.1 §6 双刀框架被外部验证。
