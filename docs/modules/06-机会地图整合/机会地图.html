机会地图 — AI × 小游戏与 Playable 体验赛道（2022-2026)#

模块 06 · 核心主交付物 最终更新：2026-04-13 本文件是 Phase 1-3 全部研究的最终整合与行动清单，服务四类读者：A 产品 PM / B 投资人 / C 行业从业者 / D 战略决策者 所有断言携带模块引用。强判断 / 推测 / 探索性假设分别标签。

§0. TL;DR(200 字）#

我一直在找一个问题的答案：如果 AI 真的会吃掉游戏，为什么 2024 Q2 微信小游戏 240+ 款季度千万流水榜单里，AI-native 作品是零？花了八个模块的研究才看清楚答案。Runtime AI 和 IAA 小游戏的交集是一块经济学真空带，GPT-4o-mini 单会话 Token 成本直接把 ARPDAU 吃穿（模块 1.1 + 3 · G06）。但 2025 年跑出来一个意外的破局形态——Generative Playable Feed(GPF）——用 Pipeline AI 配合 UGC 飞轮，把 Runtime 成本归零。这是 Phase 1-3 里我看到的唯一同时解决 Token 经济学、内容枯竭、冷启动三大痛点的路径（模块 2.5）。

真正让我意外的是中国侧的空白。Pipeline-Creator 象限全球只有 7 家玩家，中国零。12-24 个月后这块地会被 YouTube Playables Builder / Roblox / MWM 平台化吃掉，所以机会窗口是现在。最该规避的事情反而很清楚：正面 Runtime AI × IAA、纯 C 端 AI 陪伴独立公司（Character.AI $2.7B 是”失败的成功退出”）、AAA 内嵌 AI NPC（逆水寒反面）、AI-washed 叙事型公司。

一句话：GPF 是唯一真实的窗口，中国版 Pipeline-Creator 是最大空白，12-24 个月内必须下注。

§1. 主命题（深化）#

1.1 修订后的主命题#

“Runtime AI × 小游戏是真空带（Token 经济学卡死），但 Generative Playable Feed(GPF）子品类正在用 Pipeline AI + UGC 飞轮打破它 — 窗口期 12-24 个月，之后被 YouTube/TikTok/Roblox 平台化吃掉。”

1.2 为什么 Runtime 真空带成立（三重论证）#

先把数字摆上桌。GPT-4o-mini 单会话 Token 成本大约是每小时 $0.014，而小游戏 IAA 的 ARPDAU 拉不到 $0.05,AI 税在毛利里硬吃 15-40%【模块 1.1 + 1.3 + 3 · G06】。这不是执行问题，这是算术。

退出模式把这个算术盖棺定论。Character.AI 有 28M MAU，结局是被 Google $2.7B 的”失败的成功”授权收编；Inflection 被 Microsoft 以 $650M reverse acqui-hire 吸走。C 端独立 IPO 路径在 2024 正式关闭【模块 2.2】。

如果你还怀疑，看榜单。2024 Q2 微信小游戏 240+ 款季度千万流水榜里零个 AI-native 作品，TOP 1 寻道大千用 AI 的地方只有买量素材【模块 1.1 · 2.1】。

1.3 为什么 GPF 是唯一可行破局#

关键在于一个会计位置的挪动。Pipeline AI 的经济学方程式是这样跑的：AI 只在创作时被调用一次，消费端零 Token 成本；单次生成大约 $0.1-0.5，靠 creator : consumer ≥ 1 : 100 的 UGC 杠杆摊薄；消费端回归传统 IAA 或订阅，AI 税归零【模块 2.5】。

这不是省钱，是把成本项从经营性支出挪成了一次性摊销。下面这张表把每个上游模块的命题和 GPF 的对应解法放在一起——我做这张表的时候发现一件让我停了一下的事：五个模块的痛点，GPF 都命中了不同的位置。

模块 命题 GPF 如何解
1.1 双刀 AI-native × Runtime 象限红海 42 家 GPF 在 Pipeline-Creator 象限全球仅 7 家，中国零
1.3 平台 AI 税 15-40% ARPDAU Pipeline AI 把 AI 税搬到创作端
3 · G06 Token 倒挂 GPF 唯一不受 G06 影响的形态
4 · 情感 MMO D7-D30 断崖仅服务 5-10% GPF 服务轻度 95%，反面形态
5 · 定义权脱离 游戏壳 ≠ 游戏 GPF = 定义权脱离的终极形态，做 playable 但不叫游戏

1.4 双刀分类框架（本文全文复用）#

双刀是这份研究里我用得最顺的一个工具，接下来每一章都要回到它。刀 1 问的是 AI 可移除性——拆成 AI-native（移除 AI 即死）、AI-augmented(AI 让体验更好）、AI-washed(AI 只是 PR）、Enabler(AI 基础设施 B 端）。刀 2 问的是 AI 存在位置——Runtime（在线推理）、Pipeline-Dev（开发工具）、Pipeline-Creator（用户创作）、Pipeline-Embedded（素材预生成）、Hybrid。

两刀交叉出 5×4 矩阵，共 20 个格子。下一节把 92 家玩家铺进去。

§4. 确定性机会（5 个强判断）#

O1. 中国版 GPF(Generative Playable Feed)💎 P0#

服务视角:A 产品 / B 投资 / D 战略

机会描述：在抖音 / 微信 / 小红书 / B 站任一生态内，做一款”用户一句 prompt → AI 生成 playable → feed 消费 → remix 飞轮”的独立 App 或小游戏。底层用豆包 / Hunyuan / 通义千问的 pipeline 生成能力，AI 只在创作时被调用一次，消费端回到传统 IAA 或虚拟币。对标全球玩家 MWM Rezona / LoopIt / YouTube Playables Builder / Upit。差异化在中文 meme、本土 IP 和抖音式分发。这是 Pipeline-Creator × AI-native 象限全球仅 7 家、中国零玩家的空白位（模块 1.4）。

多模块佐证: - 模块 2.6 GPF 深度专题（权威）:8 家产品七字段拆解 / 27 sources / 经济学三前提推导 【S-2.6-】 - 模块 2.5 GPF 品类定义：Pipeline AI + UGC feed + creator:consumer ≥1:100 【S-2.5-】 - 模块 1.4 双刀统计：Pipeline-Creator 全球 7 家 / 中国零 【S-1.4-】 - 模块 1.3:AI 税 15-40% ARPDAU → GPF 把 AI 税搬到创作端 【S-1.3-】 - 模块 3 · G06:Token 倒挂根本解 【S-3-G06】 - 模块 5：游戏性定义权脱离的终极形态 【S-5-T1】

2.6 新增关键事实（已校正或新补）: - Decart / Oasis 2025 年 8 月 B 轮 $100M / $3.1B 估值，Sequoia + Benchmark，世界模型路线的技术极限形态 【S-2.6-06】 - YouTube Playables Builder 2025 年 12 月 beta 由 Gemini 3 驱动，限美加英澳 creator 申请审批 【S-2.6-04 / 05】 - MWM 2020 年 3 月 B 轮 5000 万欧元（Blisce 领投），2.5 节”2021 年 C 轮”为误记 【S-2.6-01】 - Mirage 2 在 Gamescom 2025 发布，Dynamics Lab 团队 <10 人，实时 UGC 游戏引擎定位 【S-2.6-11 / 12】 - Sekai $3.1M 种子（Hashed 领投 + a16z），自定义 “TikTok for mini-apps”，产品扩到 decision / poll / gift 等 mini app 【S-2.6-15 / 17】 - 反信号:miHoYo 系蔡浩宇 Anuttacon 的 Whispers from the Star 选了 Runtime AI 对话路线而非 GPF，最强 AAA 创始人看到的 AI 游戏未来是情感 LTV 而非 UGC 飞轮 【S-2.6-19 / 20 / 21】

为什么我把这条标成”确定性”：三点。第一，它同时解决 Token、内容、冷启动三大痛点，Phase 1-3 里没有第二个形态做到这件事。第二，这个品类已经有 6+ 家海外玩家在跑——MWM 双品牌、YouTube 官方、Rosebud、Upit、Replit、Websim——它不是一个假设，是一个已经被验证的品类形态。第三，字节、腾讯、米哈游在这个象限全部缺席，但招聘信号和专利信号提示他们正在筹备却还没出货（模块 2.5 中国 gap）。这三件事凑在一起，就是一个有确定时间窗的空白。

TAM 估算：若能吃到抖音小游戏大盘 10-20%，对应 ¥15-40 亿流水（基于 1.2 抖音小游戏 150-200 亿估算），上限被创作者群体规模和 feed 时长决定。

可复用路径（”四步曲”概念）: 1. 第一步·品类占位（Week 1-8）：MVP 上线，只做一类 prompt（如”生成一个 idle 放置”) 2. 第二步·Creator 杠杆（Month 3-6）：签约 50-100 个抖音腰部创作者做冷启 3. 第三步·Remix 飞轮（Month 6-12）：一键 fork + 分成，creator:consumer 拉到 1:100 4. 第四步·平台化（Month 12-24）：和抖音 / 微信官方合作，变成 SDK 层

第一步动作（Week 1-4): - W1：拉齐团队（5 人，1 全栈 + 1 prompt 工程 + 1 游戏策划 + 1 增长 + 1 BD) - W2：跑通最小 Pipeline（豆包 / Hunyuan pipeline API + HTML5 canvas 模板 10 个） - W3：做 20 个 seed playable，灰度发布在小红书 / B 站 - W4：测 creator:consumer 初始比，决定下一步模板方向

时间窗口:⚠️ 12-18 个月（P0 紧迫）。2027 年前被 YouTube Playables / Roblox / 抖音官方吃掉的概率超过 60%，这是我七成确信的判断。

O2. 游戏 AI middleware 中国版 🎯 P0#

服务视角:B 投资 / D 战略

机会描述：对标 Inworld（美国约 $500M 估值）做中国版游戏 AI middleware，打包”人设 + 长时记忆 + 声音 + 立绘 + 人格金标 eval”的 B2B SDK，同时支持 Runtime 和 Pipeline 两种集成模式。中国几乎空白，只有 Rct.ai 在转型，而 Krafton MOGIA / inZOI Smart Zoi 已经证明东亚大厂愿意付费。更锋利的版本是从”for Dev”转向”for Creator”，直接吃 Pipeline-Creator 生态位。

多模块佐证: - 模块 3 · T-1 投资主题：游戏 AI middleware 18-24 个月时间窗 【S-3-T1】 - 模块 1.4 双刀：中国 Enabler × Pipeline 近乎空白 【S-1.4-】 - 模块 2.2:Inworld / Rho / Artificial Agency / Convai 四家美国玩家，Inworld 估值 $500M 【S-2.2-】 - 模块 2.3:Krafton × Inworld 深度绑定证明大厂愿意付费 【S-2.3-】 - 模块 4：人格金标 eval 作为 Day 1 工程优先级，是 middleware 的差异化切入点 【S-4-】

为什么确定性：中国独苗空缺、美国范本已跑通、Bitkraft 和 a16z 在美国已下注过同一赛道。更关键的是 G01 / G02 / G11 三个 S1 GAP 全都是 middleware 的天然切入点，也就是说你卖的 SDK 功能清单和甲方痛点清单是重合的。

TAM 估算：中国游戏 AI middleware 2026-2028 TAM ¥5-15 亿（参考美国 middleware 总市场加中国折扣）。

可复用路径：先绑定 1 家大厂首发客户（腾讯 / 网易 / 字节 Ohayoo 任一），用首发客户的真实场景反哺 SDK，再开放社区版（Rosebud 模式）给中小开发者冷启，最后把产品卖给 Krafton / Nexon / 米哈游拓展海外。

Week 1-4:W1 Talent（从 Rct.ai / 腾讯 IEG AI Lab / 米哈游挖核心 3 人）；W2 一个 POC（人设 + 长时记忆 SDK);W3 BD 首发客户；W4 融资材料。

时间窗口：中期 18-24 个月。

O3. 端侧小模型 + WebGPU 游戏 infra 🔧 P1#

服务视角:B 投资 / C 行业 / D 战略

机会描述：做一个专门面向小游戏 / Playable Ads / HTML5 的端侧 LLM 运行时加 WebGPU 加速层，让 3B 以下的小模型在浏览器、微信 JSCore、抖音 runtime 里推理首 Token 进到 500ms 以内。这是从根本上破 G06 Token 倒挂的基础设施路径，让 Runtime AI 小游戏第一次有可能跑通。对标 Web-LLM 项目加 MLC-AI，但做游戏 SDK 特化。

多模块佐证: - 模块 3 · G06 Token 倒挂 + T5 端侧模型 【S-3-G06 · T5】 - 模块 1.3 平台：字节豆包端侧 / Apple Intelligence 2025 时间表模糊 【S-1.3-】 - 模块 2.3：日韩 AAA 清一色内部工具化验证端侧需求 【S-2.3-】

为什么确定性:G06 是整个 Runtime 真空带的根因，任何能把 Token 成本降一个数量级的方案都值钱。Apple Intelligence、豆包端侧、华为盘古端侧加在一起，打开的是 2025-2027 的硬件时间窗。模块 2.4 §9 明确把它列为 T1 + T5 + G06 的三重交集。

TAM 估算:infra 本身不大，¥1-5 亿，但卡位价值远高于直接估值。

可复用路径：开源框架冷启 → 跑通 3-5 个 showcase 游戏 → 平台授权（字节 / 腾讯）→ 变成行业默认。

Week 1-4：技术验证，重点是端侧 3B 模型 WebGPU 首 Token 延迟。

时间窗口：中长期 24-36 个月，2026 Apple Intelligence 成熟后爆发。

O4. “游戏壳 ≠ 游戏”陪伴 / 工具产品延伸 🎭 P1#

服务视角:A 产品 / B 投资 / D 战略

机会描述：复用中国星野 / 猫箱 / 筑梦岛模式，但反向做——把 Duolingo 式 Streak、每日关卡、AI 玩法核藏进陪伴、工具、教育的壳里，同时吃到订阅制 ARPU(3-5 倍 IAP）、绕苹果税、绕中国版号、绕监管红利。具体切面包括 AI 口语配关卡化、AI 心理咨询配任务感、AI 家教配剧情、AI 角色扮演配成长系统。真正的诀窍是把游戏玩家心理学的 Streak、损失厌恶、compulsion loop 装进非游戏品类。

多模块佐证: - 模块 2.1：星野 2024 流水 > 3 亿，付费率 3-5%，头部月付 ¥1000+ 【S-2.1-】 - 模块 5 Top 5 #5 Streak + 损失厌恶空白蓝海 【S-5-T5】 - 模块 5：游戏性定义权脱离最锋利洞察 【S-5-】 - 模块 4:Replika Pro Churn < 3% 证明订阅心智可被情感锁死 【S-4-*】 - 模块 2.4 T-3:”游戏壳 ≠ 游戏”陪伴 / 工具产品

为什么确定性：豆包 / 猫箱 / 星野已经在无意识地做这件事，但没人把它写成方法论，先入场者建立心智。订阅制 ARPU 是 IAP 的 3-5 倍，再叠版号规避的三重红利。模块 2.4 §9 A-3 明确把它列为确定性机会。

TAM 估算:AI 陪伴与 AI 教育交集的中国市场 ¥20-50 亿（2027）。

可复用路径：垂直场景切入（口语最明确）→ 一个玩法内核通吃 → 扩品类。

Week 1-4：选 1 个垂直（口语最明确），跑最小 MVP，测 D7 留存。

⚠️ 监管风险:Garcia v. Character Technologies 判决是行业最大监管不确定性 【S-4-*】。

时间窗口：短中期 12-24 个月。

O5. AAA Runtime AI 合作 / 授权（Krafton MOGIA × Inworld 模式）🤝 P1#

服务视角:C 行业 / D 战略

机会描述：不自己做 C 端，而是做 AAA 大厂的 AI NPC 技术授权方。Krafton × Inworld 是东亚大厂第一个把 AI NPC 作为”用户可见卖点”商业化上线的案例（inZOI 2025.03 EA）。同一逻辑可以复制到米哈游、腾讯 IEG、网易、Nexon、米哈游海外工作室。关键动作是做一个”看上去像产品但本质是 demo”的 showcase，然后技术授权收费。

多模块佐证: - 模块 2.3:Krafton × Inworld 深度绑定 · inZOI 2025.03 EA 商业化上线 【S-2.3-】 - 模块 2.1：逆水寒 AI NPC 反面案例（G01 / G04) 【S-2.1-】 - 模块 3 · T-1:middleware 投资主题 【S-3-T1】

为什么确定性：东亚第一个跑通的 Runtime AI 商业案例已经出现；大厂内部工具化惯性强，授权比自研快；2026-2027 韩国 Lab 集中爆发的时间窗还在打开 【S-2.3-*】。

TAM 估算：全球 AAA 游戏 AI NPC 授权市场 $200-500M（2028）。

可复用路径:1 款 showcase 游戏 → 技术服务收费 → 并购退出给大厂。

Week 1-4：展示版 demo 立项加 BD 1 家大厂 PoC 意向。

时间窗口：中期 18-30 个月。

§5. 探索性机会（4 个弱判断）#

标签:🧪 探索性假设 — 未被多模块数据完整验证，高回报高风险

下面这四条我没有完全信。它们的回报都够大，但证据链里都有一两个关键缺口。我列出来是因为它们值得追踪，不是因为现在就该下注。

E1. Telegram Mini App × AI(Catizen 路径）🧪#

服务视角:A / B - 假设:Catizen / Hamster Kombat 证明 Telegram Mini App 的 IAA + Web3 经济学能跑，若叠加 Pipeline AI 生成内容，可复制 Mini App × AI 奇点 - 缺口:Catizen 真实 CPM / ARPDAU 未公开（2.4 §8 待补） - 第一步：做 1 个 Mini App MVP 测 3 日留存 - 为何弱判断:Telegram 生态是否稳定、中国团队出海是否能吃到 Web3 红利两大未知

E2. 订阅制 AI 陪伴 × 教育壳 🧪#

服务视角:A / D - 假设：把 AI 家教 / AI 口语和 Streak 游戏化深度融合，服务 K12 海外市场（避开国内教培监管） - 第一步：选 ESL 口语做 MVP，测订阅价格点 - 为何弱判断:K12 监管加付费意愿在不同国家差异巨大

E3. 日本 VN × AI 破局（低概率高回报）🧪#

服务视角:C / D - 假设：找 1 家日本中型 Galgame 厂（Marvelous / FuRyu / Visual Arts / Key）授权 IP 做 AI 化续作，用”粉丝续命”叙事破 VN 作者性锁死 - 缺口：日本 Galgame 中型厂 AI 动作几乎零公开信息（2.4 §8 gap) - 为何弱判断：日本 VN 作者性和 IP 法务结构系统性阻止这件事（2.3 明确判断），成功率低于 10%，但一旦跑通是独门独路

E4. 人格金标 eval SaaS 🧪#

服务视角:B - 假设：模块 4 强烈建议 “人格金标 eval” 作为 Day 1 工程优先级，这是一个尚未出现独立玩家的细分 - 第一步：做一个 eval 工具，免费给 3 家 AI 陪伴公司试用 - 为何弱判断:Tooling 赛道的客户基数是否足够支撑独立 SaaS 不清楚
