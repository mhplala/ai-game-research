06B. 投资机会 — 给一级市场 VC / 战投 / 二级研究员的赛道地图#

视角:VC / 战略投资部 / 产业投资基金 / 二级市场研究员 核心问题:”这个赛道我该投什么？估值多少？退出路径怎么走？” 最后更新:2026-04-13

0. TL;DR（给投资人的 30 秒版本）#

一句话判断:

“2022-2023 的 C 端 AI-native 已过峰值，2024H2 起资本全面转向 B2B middleware 和 GPF 新物种 — 2025-2027 的三大主题是 GPF 创作工具 / AI NPC middleware / 端侧小模型 infra。

关键警示：Character.AI $2.7B 技术授权是”失败的成功退出”，老股东实得远低 10x。纯 C 端 AI 陪伴独立 IPO 路径已关闭。“

三大投资主题:

主题 时间窗 阶段 估值区间（典型） 核心风险 推荐度
T1. GPF 创作工具（中国版） 12-18 个月 Seed-A $5-30M 平台先发占位 ⭐⭐⭐⭐⭐
T2. AI NPC middleware（中国版） 2 年 Seed-B $10-50M 海外 Inworld 先发 ⭐⭐⭐⭐⭐
T3. 端侧小模型 × 游戏 infra 2-3 年 Seed-A $5-20M 时机早 ⭐⭐⭐⭐
T4. GPF 创作工具（海外） 已进行 A-B $20-200M MWM / YouTube 抢 ⭐⭐⭐
T5. AI-native 小而美 Indie 持续 天使 $1-5M 筛选困难 ⭐⭐⭐
T6. 纯 C 端 AI 陪伴独立公司 已关闭 — — 天花板 $20-40 LTV ❌

1. 赛道温度图：2024H2 分化#

资本流向（2022-2026)#

年 AI-native 融资 主要去向
2022 $200M+ Character.AI / Inflection / Latitude / Replika
2023 $450M 峰值 Inworld / Hidden Door / Altera / Rho / a16z Games 密集下注
2024 H1 ~$250M 主题仍在 C 端游戏，但开始看 middleware
2024 H2 ~$150M（回落 30%) 全面转向 B2B middleware
2025 预计 ~$300M middleware + GPF + 端侧 infra

资本逻辑：从”我要做下一个 Blizzard” → “我要做下一个 Unity / Inworld / Adobe for Games”

2024H2 两大转折事件（投资人必读）#

  1. Character.AI → Google $2.7B 技术授权(2024.08) - Shazeer + De Freitas 回流 Google - 28M MAU 但 Token vs ARPDAU 跑不通 - a16z 实得远低 10x，这是一个失败的”成功退出” - 意义：定义了纯 C 端陪伴独立 IPO 路径关闭

  2. Inflection → Microsoft $650M(2024.03) - Suleyman 带队加入 MS - 同构 reverse acqui-hire - 意义:reverse acqui-hire 成为 AI 陪伴创业的唯一顶部出口

2. 三大投资主题详细拆解#

T1. GPF 创作工具（中国版）⭐⭐⭐⭐⭐ 最强推荐#

核心逻辑: - Pipeline-Creator 象限全球仅 7 家（模块 1.4 数据） - 中国市场完全空白，只有字节 / 腾讯 / 快手在底层 SDK 层面有动作，没有终端 Creator 工具 - 12-18 个月窗口期，之后被 MWM / YouTube Playables Builder / Roblox 抢走 - Pipeline AI 绕开 Token 经济学真空带 → 唯一能在”AI × 小游戏”交集里真实赚钱的形态

海外对标: - Rezona / LoopIt(MWM.ai，法国 C 轮 $50M / ~200 员工） - YouTube Playables Builder(Google，平台级） - Upit / Jabali / Rosebud / Websim（小而多）

典型项目画像: - 团队：3-8 人（1-2 产品 + 2-3 LLM 工程 + 2-3 游戏引擎 + 1 社区运营） - 启动资金：¥500 万 - ¥1500 万(Seed) - 6-12 个月 MVP（抖音小游戏 / 微信小游戏 / H5) - 1-2 年冲 500 万 MAU - A 轮估值区间 $10-30M

退出路径: - 被字节 / 腾讯 / 快手收购（最可能，新业务缺口） - 海外 Series B 融资（Rezona 级别） - Google Playables Builder 海外上市后的联动并购

风险与规避: - 最大风险：字节 / 腾讯自己下场做同类产品 → 规避：和豆包 / Hunyuan 深度绑定做生态位玩家，而非竞争者 - 次大风险：平台政策变化（如微信小游戏禁止 UGC)→ 规避：多平台分发

投资推荐阶段:Seed 到 A 轮

值得关注的信号: - MWM.ai Rezona DAU 40 万（来自模块 06A agent 的发现）— 用这个作为中国版 GPF 的 PMF 参照

T2. AI NPC Middleware（中国版）⭐⭐⭐⭐⭐#

核心逻辑: - 美国 Inworld AI 估值 $500M+ / 客户包括 Krafton / NetEase Miaozi / Niantic - 中国 Pipeline-Dev 象限除 rct AI 外几乎空白 - 模块 3 G01（长时记忆）+ G02（人格一致性）+ G04(Token 成本）是所有 AI 游戏开发者的共同痛点 - 中国 AAA 大厂（腾讯 / 网易 / 米哈游）需要 middleware 但买不到中文版

对标: - Inworld AI（美国，$500M,AAA 客户） - Convai（美国 $10M seed) - Series AI / Rho（美国 $28M Series A) - Kinetix（法国 AI 动作生成） - rct AI / Chaos Box（中国，转型中）

典型项目画像: - 团队：10-30 人（5-10 LLM 工程 + 3-5 游戏引擎 + 3-5 MLOps + BD + 医学… 啊不对 + 开发者关系） - 启动资金：¥1500 万 - ¥5000 万(Seed 到 A) - 核心产品：打包”人设 + 记忆 + 声音 + 立绘 + 审核“一站式 SDK - 18-24 个月拿到第一个 AAA 客户（腾讯光子 / 米哈游 / 网易）

退出路径: - 中国大厂收购（腾讯 / 字节 / 米哈游战投） - Series B 估值区间 $50-200M - 长期做到”中国 Inworld” 状态，$500M 估值

最值得投的角度: - 已有 rct AI 在转型，但尚未形成壁垒 - 腾讯 Hunyuan Game 目前是 Lab 状态，没有产品化 - 3 年内 2-3 家中国 middleware 有机会跑出 — 现在投的性价比最高

T3. 端侧小模型 × 游戏 infra ⭐⭐⭐⭐#

核心逻辑: - Token 真空带的根本破法 — 把 LLM 推理从云端搬到端侧，成本归零 - Llama 3.2-3B / Qwen 2.5-3B / Phi-4-mini 已经能在手机端跑 - WebGPU + Web LLM 已经能在浏览器里跑 ~100 tokens/sec - 2026 H2 是 Apple Intelligence SDK 广泛开放的关键节点

对标: - Web LLM(Chromium 生态） - Apple Core ML + MLX(Apple 生态） - Qualcomm AI Hub(Android 生态） - MLC LLM / llama.cpp WebGPU port

典型项目: - “Web LLM for Games” infra 公司：为 Unity / Godot / Phaser / PixiJS 提供端侧 LLM SDK - “Apple Intelligence for Game Dev” 工具链 - “WebGPU Playable Engine”：专门为 GPF / 小游戏优化的 WebGPU 游戏引擎

风险: - 时机可能早 6-12 个月（端侧 3B 模型能力仍有限） - 平台（Apple / Google）可能自己做，把创业公司挤死

投资推荐：天使到 Seed，赌未来 24 个月

T4. GPF 创作工具（海外）⭐⭐⭐#

  • Rezona / LoopIt(MWM.ai 已到 C 轮）
  • Upit（早期阶段）
  • Jabali.ai(Africa 团队，早期）
  • Rosebud AI(~$4M seed)

投资建议:MWM 已到后期，估值不便宜；Upit / Jabali / Rosebud 仍是 Seed 阶段值得小投。

T5. AI-native 小而美 Indie（天使）⭐⭐⭐#

核心逻辑:Suck Up! < 5 人团队 / Steam 爆款 / $9.99 买断证明”小而美“可跑通。Infinite Craft solo 零融资 Poki 现象级证明”极简玩法 + 低频 LLM 调用 + 广告变现“可跑通。

投资策略: - 单笔小额（$50-500K）天使投 10-20 个 team - 接受 80% 失败率，10% 回本，10% 翻 10 倍以上（一个 Suck Up! 级爆款就 cover 了） - 筛选指标:team 背景（Steam 上架经验 / 独立开发者社区声誉）、Demo 可玩性

T6. 纯 C 端 AI 陪伴独立公司 ❌ 已关闭#

为什么不投: - Character.AI $2.7B 技术授权 = a16z 实得远低 10x - Inflection $650M = 同构的失败退出 - LTV 天花板 $20-40（被模块 4 数据证实） - 情感 MMO 5-10% 付费率（模块 4）决定了总 TAM 不够大

例外：大厂内部项目 / 分拆（米哈游 Anuttacon 模式）可以考虑。

3. 区域投资地图#

美国#

  • middleware 已是红海(Inworld 领跑，Convai / Rho 次席）— 晚期估值泡沫
  • 新物种：GPF 创作工具 早期估值合理（Upit / Jabali / Rosebud)
  • 端侧小模型 infra 早期
  • AI-native Indie 天使阶段

推荐：以 middleware 晚期收购 / Indie 天使为主

中国#

  • middleware 几乎空白 — 黄金赛道,2 年内 2-3 家跑出
  • GPF 创作工具 零玩家 — 金矿,12-18 个月窗口
  • AI 陪伴已有头部（星野 / 猫箱 / 筑梦岛）— 拼场景细分
  • AI-native Indie 生态弱 — 相对难投

推荐：以 middleware 孵化 / GPF 创作工具 Seed 为主

日本#

  • AAA 内部工具化，不对外 — 没有独立投资标的
  • VN 土壤结构性锁死 — 作者性和 IP 法务阻碍
  • 建议：不建议单独投日本，可作为美国项目的亚洲扩张节点

韩国#

  • 4 大 Lab(MOGIA / VARCO / Nexon IL / Smilegate)都是大厂内部，不对外
  • ScatterLab Zeta / Luda 有独立投资可能
  • 2026-2027 爆发窗口 — 现在投可能早

推荐：关注 Zeta 等独立公司，或通过 Krafton 战投部合作

5. 退出路径分析#

美国（模式成熟）#

  1. Acqui-hire(Character.AI / Inflection 模式）— 大厂买技术 + 人才
  2. Strategic 并购（未来可能 Meta / Google / Microsoft 收 middleware)
  3. Series B/C 大额融资后继续独立经营(Inworld 路径）
  4. Steam 买断爆款(Suck Up! 这类不融资的纯现金流 model)

中国（模式分化）#

  1. 巨头收购（猫箱 → 字节、星野 → MiniMax）— 但被巨头内部合并，不是独立退出
  2. IPO — 基本不现实(AI 游戏 IPO 还没先例）
  3. A 股借壳 — 理论可行但实际罕见
  4. 卖给 A 股游戏公司（三七 / 巨人 / 完美 / 吉比特）— 可能性低

警示：中国市场退出路径结构性缺失#

  • 美国有 reverse acqui-hire 顶部出口，中国没有
  • 美国有 middleware $500M 估值先例(Inworld），中国无
  • 中国头部 AI 产品（星野/猫箱）绑定巨头内生，没有独立上市
  • → 对中国 VC 而言，退出路径靠”大厂收购” 一条，流动性风险高

7. 给 LP / 基金管理人的配置建议#

1 年内立刻配（60% 仓位）#

  • T1 GPF 中国版 1-2 个标的，共 2000-5000 万
  • T2 AI NPC middleware 中国版 1-2 个标的，共 3000-8000 万
  • T5 AI-native Indie 5-10 个天使，共 2000-5000 万

2-3 年布局（30% 仓位）#

  • T3 端侧小模型 infra 1-2 个标的
  • T4 GPF 海外 1-2 个标的（Upit / Jabali / Rosebud 等 Seed)
  • Krafton MOGIA 模式跨境合作 1 个

观察不投（10%)#

  • 纯 C 端 AI 陪伴
  • AAA AI NPC 工具（等 Inworld 格局明朗）
  • 日本 Galgame × AI（结构性锁死）
