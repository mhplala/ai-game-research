## 2. 研究的核心命题(最重要的一段)

### 原命题 vs 修订命题

**原命题**(研究方案 v1.0):
> "AI 如何重塑小游戏与 playable 体验赛道?"

**研究中期发现的修订命题**:
> **"Runtime AI × 小游戏是真空带(Token 经济学卡死),但 Generative Playable Feed(GPF)子品类正在用 Pipeline AI + UGC 飞轮打破它 — 窗口期 12-24 个月,之后被 YouTube / TikTok / Roblox 平台化吃掉。"**

### 三大命题支柱

#### 1. 经济学真空带(模块 1.1 + 1.3 + 3)

- **Token/session $0.014**(GPT-4o-mini)
- **IAA ARPDAU < $0.05**
- 毛利空间根本不够支付 AI 税
- AI 税吞掉 **15-40% 的 ARPDAU**(模块 1.3)
- **G06 Token 倒挂** 是 18 条 GAP 中最硬的墙(模块 3)

#### 2. 游戏性定义权脱离(模块 5 + 2.1)

- Character.AI / 星野 / 猫箱 / Duolingo Max / Midjourney Discord **都不叫自己游戏,但行为循环完全同构**
- **"游戏性"的定义权已经从游戏行业脱离**
- 中国版号强制 + 美国自发选择 + 日韩文化偏好 = 三地同一结论
- **未来最优形态不是做游戏,而是把游戏核藏进陪伴 / 工具 / 教育的壳**

#### 3. Pipeline AI 新物种(模块 2.5 GPF)

- **Rezona / LoopIt**(MWM.ai 法国)
- **YouTube Playables Builder**(Google 2024-2025)
- **Upit / Jabali.ai / Rosebud / Websim**
- 共同机制:**text prompt → AI 生成 playable → TikTok 式 feed → 一键 remix**
- **经济学公式**:AI 只在创作时调用(~$0.1-0.5),玩家消费静态生成结果,**Runtime AI 成本 = 0**
- **同时解决 Token 经济学 + 内容枯竭 + 冷启动三大痛点**

### 三个最锋利的象限发现

1. **"跑通比例" 只有 6/85 ≈ 7%** — 整个赛道仍在高度不确定期
2. **赛道两端分化**:AAA Runtime AI(Krafton MOGIA / inZOI)首次跑通 vs 中型创业公司(Hidden Door / AI Dungeon)集体失血
3. **💥 中国 Pipeline-Creator 象限零玩家** — 腾讯 / 字节只做底层 SDK,12-24 个月内有被 MWM / YouTube / Roblox 先发占位的风险

## 8. 研究发现的独特价值(相比同类报告)

1. **"AI 可移除性刀" + "AI 存在位置刀" 双刀分类框架** — 精准定位 Runtime × IAA 真空带和 Pipeline × UGC 蓝海
2. **修订主命题的能力** — 研究中期发现 GPF 新物种后能快速调整 scope
3. **四视角交叉** — 同一份素材服务 A/B/C/D 四种读者
4. **92 家玩家全景图 + 双刀象限统计** — 发现 Pipeline-Creator × 中国零玩家
5. **"游戏性定义权脱离"的中国版号强化**解读 — 中国监管反向助推创新的独特角度
6. **Character.AI → Google $2.7B 的正确解读** — 不是胜利是失败("失败的成功退出")
7. **反直觉的玩家心理洞察**:A/B 测试在 AI 游戏里有毒 / D1/D7 KPI 误导 / grief 退坑 / Token 不可视化
8. **完整覆盖 2025 Q3-Q4 新物种**:GPF / Rezona / LoopIt / YouTube Playables Builder
